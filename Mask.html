1. 프로젝트 개요

  • 프로젝트명: 마스크 착용 상태 인식 시스템 (YOLOv5 기반)
  • 개발 기간: 2023.10 – 2023.12
  • 참여 인원: 1명
  • 나의 역할: 전체 기획, 데이터 전처리, 모델 학습 및 평가, 커스텀 데이터셋 구축, 결과 시각화 및 보고서 작성

2. 개발 환경 및 기술 스택

  • 사용 언어: Python
  • 프레임워크: PyTorch, YOLOv5
  • 개발 도구: Google Colab, Jupyter Notebook, OpenCV, Pandas, Matplotlib, Seaborn

3. 데이터 구성 및 전처리

라벨은 Pascal VOC 포맷(XML)으로 주어졌으며, YOLO 포맷(.txt)으로 자동 변환하였습니다. 이미지 크기에 맞춰 bounding box를 재계산하고 클래스 ID를 유지한 채 데이터셋을 train/val/test로 분리하였습니다.

4. 모델 학습 및 하이퍼파라미터

  • 모델 아키텍처: YOLOv5s (yolov5l은 과적합 경향)
  • Epochs: 50
  • Batch size: 16
  • Learning rate: 0.01

하이퍼파라미터 튜닝을 통해 모델의 안정성과 성능을 확보했습니다.

5. 성능 평가 및 분석

  • 정확도 (mAP@0.5): 86%
  • Precision: 0.88 / Recall: 0.85
  • 부적절 착용 케이스는 데이터 부족으로 오탐 가능성이 존재

전체적으로는 높은 탐지율을 보였으며, 클래스 간 성능 차이를 확인했습니다.

6. 탐지 결과

YOLOv5를 활용한 마스크 탐지 학습 및 정확도 탐지 그리고 결과입니다.

7. 문제 해결 사례

  • 직면한 문제: XML 어노테이션 → YOLO 포맷 변환 시 좌표 오차 발생
  • 해결 과정: 이미지 시각화를 통해 직접 수식 검증 및 보정
  • 학습점: 객체 탐지 라벨 구조에 대한 실전 감각 습득
