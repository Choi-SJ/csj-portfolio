건물 에너지 효율 회귀 예측

← 포트폴리오 홈으로

1. 프로젝트 개요

  • 프로젝트명: 건물 Cooling Load 예측 모델 구현
  • 개발 기간: 2025.04
  • 참여 인원: 1명
  • 나의 역할: 데이터 분석, 모델 구축, 성능 비교 및 리포트 작성

2. 데이터셋 및 전처리

  • 출처: Energy Efficiency Dataset (Kaggle)
  • 구성: 768개 샘플, 8개 특성
  • 타겟 변수: Cooling Load (Y2)
  • 전처리: 결측치 없음, 수치형 입력, StandardScaler로 정규화

3. 모델 구축 및 학습

  • 사용 모델: Linear Regression, RandomForestRegressor
  • 데이터 분할: 8:2 (train/test)
  • 스케일링: StandardScaler

4. 성능 평가

  • 평가지표: RMSE, MAE, R² Score
  • 결과 시각화: 예측 vs 실제 산점도, ROC Curve
  • RandomForest 모델은 R² ≈ 0.99로 우수한 성능

5. 하이퍼파라미터 튜닝

GridSearchCV로 최적 조합을 탐색한 결과, 다음 설정에서 성능 향상을 확인했습니다:

  • n_estimators: 200
  • max_depth: 10
  • min_samples_split: 2

6. 프로젝트 고찰

  • 장점: 데이터가 깨끗하고 예측 정확도가 높음
  • 한계: 데이터 다양성 부족으로 일반화 성능 검증은 제한적
  • 활용: 건축 설계 단계에서 냉방 수요 예측 도구로 활용 가능
