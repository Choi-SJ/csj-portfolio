버섯 분류 모델 (RandomForest 기반)

← 포트폴리오 홈으로

1. 프로젝트 개요

  • 프로젝트명: 버섯 분류 모델 구현 및 분석
  • 개발 기간: 2025.04
  • 참여 인원: 1명
  • 나의 역할: 데이터 전처리, 모델 개발, 성능 분석, 리포트 작성

2. 데이터셋 및 전처리

  • 출처: Kaggle Mushroom Dataset
  • 크기: 8124개 샘플 × 22개 변수
  • Target: Mushroom_quality (e: 식용, p: 독성)
  • 전처리: '?' 결측값 제거, One-Hot Encoding 적용

3. 모델 구축 및 학습

  • 사용 모델: RandomForestClassifier (기본 및 튜닝 버전)
  • 분할 방식: train_test_split (80:20)
  • 튜닝: GridSearchCV로 n_estimators, max_depth, min_samples_split 최적화

4. 성능 평가

  • Accuracy: 100%
  • Precision / Recall / F1-score: 모두 1.0
  • ROC-AUC: 1.0
  • Confusion Matrix, ROC Curve 시각화

5. 하이퍼파라미터 튜닝

GridSearchCV를 사용해 다음과 같은 조합을 탐색하였고, 최적의 성능을 유지하며 검증할 수 있었습니다.

  • n_estimators: [100, 200]
  • max_depth: [5, 10, None]
  • min_samples_split: [2, 5, 10]

6. 프로젝트 고찰

  • 장점: 높은 정확도, 빠른 학습, 전처리 부담 적음
  • 한계: 실생활 데이터처럼 잡음이 많은 환경에는 추가 검증 필요
  • 활용: 생물학적 샘플 자동 판별 시스템 등
